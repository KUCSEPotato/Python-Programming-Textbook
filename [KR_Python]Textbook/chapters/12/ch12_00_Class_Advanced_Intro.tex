\chapter{클래스 (Class) — 심화}

이 장에서는 클래스의 기본 문법을 넘어,
파이썬 클래스가 실제로 어떻게 동작하는지,
그리고 더 큰 프로그램에서 클래스를 안전하고 확장 가능하게 설계하려면
무엇을 알아야 하는지를 다룹니다.

특히 다음 주제들은 서로 강하게 연결되어 있습니다.

\begin{itemize}
  \item 이름(name)은 값(value)을 담는 상자가 아니라, \textbf{객체(object)}를 가리키는 연결이다
  \item 스코프(scope)와 이름 공간(namespace)이 이름 검색을 결정한다
  \item 클래스 정의는 독특한 방식으로 \textbf{새 이름 공간}을 만든다
  \item 상속과 다중 상속은 어트리뷰트 검색 규칙을 확장한다
  \item 이터레이터/제너레이터는 클래스에 ``for문 동작''을 부여한다
\end{itemize}
