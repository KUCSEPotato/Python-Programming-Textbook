\chapter{조건문(Conditional Statement)}

프로그래밍을 하다 보면 특정 조건에 따라
프로그램의 실행 흐름을 다르게 제어해야 하는 상황이 자주 발생합니다.
이처럼 조건에 따라 서로 다른 동작을 수행하도록 만드는 문법을
\textbf{조건문(Conditional Statement)}이라고 합니다.

Python에서 조건문을 구현하는 대표적인 방법은
\texttt{if} 문이며,
Python 3.10 이후부터는
여러 경우를 분기 처리할 수 있는
\texttt{match} 문도 사용할 수 있습니다.

이들은 다른 언어들에서 발견되는 \texttt{switch} 나 \texttt{case} 문을 대신합니다.
